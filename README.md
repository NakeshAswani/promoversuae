# 🚚 Promovers UAE – Logistics Website

A static logistics and relocation service website built using HTML and CSS, focused on clean layout design and professional presentation.

## 🌐 Live Demo
https://promoversuae.vercel.app/

## 🧰 Tech Stack
- HTML
- CSS

## ✨ Features
- Professional logistics website layout
- Responsive design for multiple screen sizes
- Well-structured sections (Services, About, Contact)
- Clean and user-friendly interface
- Consistent design and spacing

## 🧠 Project Overview
This project is a frontend logistics service website created to practice layout structuring and responsive UI design using pure HTML and CSS. The goal was to build a clean and professional business website suitable for a service-based company.

## 📥 Installation & Setup
1. Clone the repository  
    ```bash
    git clone https://github.com/NakeshAswani/promoversuae.git
    ```
2. Navigate to the project directory
    ```bash
    cd promoversuae
    ```
3. Open the project
    ```bash
    Open index.html in your browser
    ```
